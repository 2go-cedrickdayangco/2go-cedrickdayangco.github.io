Request Parameters
Name Type Required Description
tracking_no String(128) True Unique Tracking No / AWB Number. Reference Number
customer_account_no String(128) True Customer Account Number. NEWEXP for express walkin, NEWLCLREG for LCL walkin. Must exist in Merchant master data.
cust_short_name String(128) True Customer Short Name
dest_store_port String(128) Optional Destination Port of Store
store_id String(128) Optional Fusion Store Location ID. Must exist in Location master data
pickup_date Date
YYYY-MM-DD		 True If pickup type is "Field", input the preferred pickup schedule. If pickup_type is "Drop Off", input date of when the pickup happened
pickup_type String(128) True Field - Request for Pickup
Drop Off - Already On-Hand
dest_system String(128) True Qair or Zip
Will still be available in Phase 2?
paymode String(128) True PRP - Prepaid
FC - Freight Collect
CS - Collect Shipper
CC - Collect Consignee
SC - Service Cargo
booking_no String(128) Optional NPR Booking Number
service_type String(128) True PP - Pier to Pier
PD - Pier to Door
DD - Door to Door
DP - Door to Pier
AA - Airport to Airport
AD - Airport to Door
DDS - Door to Door Same Day
ADS - Airport to Door Same Day
shipper_name String(128) True Shipper name in Merchant Master
shipper_contact String(128) Optional Shipper Contact Number
shipper_address String(500) True Shipper Address
shipper_area_code String(128) True 2GO Area Code based on the Shipper Address
origin_landmark String(500) Optional Pickup Location Landmark
ultimate_shipper String(128) Optional Name of the sender if different from the Shipper Name
ultimate_shipper_contact String(128) Optional Contact Number of the ultimate shipper
ultimate_shipper_address String(500) Optional Ultimate Shipper Address
ultimate_shipper_area_code String(128) Optional 2GO Area Code based on the Ultimate Shipper Address
origin_port String(128) True Origin Port. Must exist in Port master data.
cnee_name String(128) True Consignee Name
cnee_contact String(128) Optional Consignee Contact Number
cnee_city String(128) Optional Consingee Address
cnee_brgy String(128) Optional Consingee Address
cnee_address String(500) True Consingee Address
cnee_area_code String(128) True 2GO Area Code based on the Consignee Address
ultimate_cnee String(128) Optional Name of the receiver in case the consignee is not available during delivery
dest_port String(128) True Destination Port. Must exist in Port master data.
dest_landmark String(500) Optional Consignee Address Landmark
cnee_careoff_store String(128) Optional Consignee CO (2GO Store). 2GO Store ID where the shipment should be picked-up by Consignee
return_cnee_name String(128) Optional Name of the receiver for return shipment. If return details is different from origin details
return_cnee_contact String(128) Optional Contact Number of the receiver for return shipment. If return details is different from origin details
return_cnee_address String(128) Optional Return Address. If return details is different from origin details
return_cnee_city String(128) Optional Return Address City. If return details is different from origin details
return_cnee_port String(128) Optional Return Address Port. If return details is different from origin details
return_cnee_area_code String(128) Optional Return Area Code. If return details is different from origin details
declared_value Decimal(12,2) True Declared Value of the shipments
cod_amount Decimal(12,2) True Amount to be collected upon delivery
inbounding_hub String(128) Optional Inbounding Hub
special_instruction String(500) Optional Free text field
attachment String(128) Optional Free text field
or_number String(128) Optional Official Receipt Number. Required if paymode is Prepaid
or_amount Decimal(12,2) Optional Official Receipt Amount. Required if paymode is Prepaid
or_date Date
YYYY-MM-DD		 True Date of when the Official Receipt was issued. Required if paymode is Prepaid
mps_flg (Y/N) True Flag to indicate if the shipments is Multi-Part Shipment(MPS)
item_details Array True Parent wrapper for the item details information
item_no Number True Item Number. Sequencial and must start to "1"
item_description String(128) Optional Item Description
block_measurement (Y/N) Optional Flag to indicate if the shipments have block measurement
item_qty Number True Item Quanity
package_code String(128) True Fusion Package Code
commodity_code String(128) True Commodity Code must be present in Commodity master list
item_weight Decimal(12,2) True Item Actual Weight in KG
item_length Decimal(12,2) True Item Length in cm
item_width Decimal(12,2) True Item Width in cm
item_height Decimal(12,2) True Item Height in cm
