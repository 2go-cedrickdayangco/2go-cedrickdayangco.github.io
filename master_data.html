Master Data API
Change Logs
Version Description
v.20221012 Initial Document
Description
Collection of FarEye API that enable 2GO System to query master data from FarEye.

Customer Master API

Endpoint URL
GET /masterdata/merchant
Response Parameters
Name Type Required Description
merchant_code String(128) True FarEye Merchant Code
merchant_legal_entity String(128) True FarEye Merchant Legal Entity
billtosite_account String(128) True Fusion Bill-to-site Account Number
merchant_address String(128) True Merchant Address
payment_term String(128) Optional Payment Term : credit, cash
contact_number String(128) Optional Merchant Contact Number
credit_line_status String(128) Optional Credit Line Status : Activate
cod_flag String(128) Optional COD Flag if merchant is allowed for COD transaction
customer_shortname String(128) Optional Customer Short Name

Get Hubs
Endpoint URL
GET /masterdata/hubs?province={province_name}&city={city_name}
Response Parameters
Name Type Required Description

port_code String(128) True Port Code
delivery_category String(128) True Hub serviceability : SA, OTD, OSA
del_lead_time_air Number True Delivery Lead Time for Air

Store Location API

Endpoint URL
GET /masterdata/store_location
Response Parameters
Name Type Required Description
location_code String(128) True Store Location Code
location_name String(128) True Store Location Name
location_address String(128) True Store Location Address
location_contact_person String(128) True Store Location Contact Person
location_contact_number String(128) True Store Location Contact Number
location_city String(128) True Store Location City
location_type String(128) True Store Location Type : 2GO Store Owned, 2GO Store Agent
location_hub String(128) True Location Hub
holding_period Number True Holding Perion
postal_code String(128) Optional Postal Code
acceptance_hub_flg String(128) Optional Acceptance Hub Flag
cartage_fee_flg String(128) Optional Cartage Fee Flag
latitude Decimal(12,2) Optional Store Location Latitude
longitude Decimal(12,2) Optional Store Location Longitude
sea_flg String(128) Optional Sea Flag
fusion_vendor_code String(128) Optional Fusion Vendor Code (for Agents only)
fusion_vendor_legal_entity String(128) Optional Fusion Vendor Legal Entity (for Agents only)
location_port_code String(128) True Store Location Port Code
pickup_store_flg String(128) Optional Pickup Store Flag
2go_store_flg String(128) Optional 2GO Store Flag
outgoing_commission String(128) Optional Outgoing Commission
incoming_commission String(128) Optional Incoming Commission
email_address String(128) Optional Store Email Address
