Request Parameters
Name Type Required Description
tracking_no String(128) True Shipment AWB form number
adjustment_number String(128) True Adjustment Reference Number
adjustment_datetime Datetime
(YYYY-MM-DD HH:mm:ss)		 True Datetime when the adjustment is done
adjustment_type String(128) True Type of Adjustment
adjustment_remarks String(128) Optional Adjustment remarks
updated_by String(128) True User who made the adjustment
billtosite_account String(128) True Fusion Bill To Site Account Number on where the charges should be accounted.
cust_short_name String(128) True Customer Short Name must be existing in the Merchant Master.
shipper_billtosite_account String(128) Optional Shipper Fusion Account Number-Bill To unique Combination
shipper_name String(128) True Legal Entity Name
cnee_billtosite_account String(128) Optional Consignee Fusion Account Number-Bill To unique Combination
cnee_name String(128) Optional Consignee Name
dest_port String(128) Optional Updated Port of Destination.
dest_area_code String(128) Optional Updated Destination Area Code
service_type String(128) Optional Updated Service Type
paymode String(128) Optional Updated Pay mode
cod_amount_old Decimal(12,2) Optional Original COD Amount to be collected
cod_amount_final Decimal(12,2) Optional Final COD Amount
declared_value_old Decimal(12,2) Optional Original Declared Value
declared_value_final Decimal(12,2) Optional Final Declared Value
pullout_fee Decimal(12,2) Optional For OR Refund / Pull-out Fee
delivery_charge Decimal(12,2) Optional
otd_fee Decimal(12,2) Optional Out of Town Delivery Fee
osa_fee Decimal(12,2) Optional Out of Serviceable Area Fee
fuel_surcharge Decimal(12,2) Optional Fuel Surcharge
bunker_surcharge Decimal(12,2) Optional
perishable_fee Decimal(12,2) Optional
transshipment_fee Decimal(12,2) Optional
bsa_fee Decimal(12,2) Optional Block Space Agreement
valuation_fee Decimal(12,2) Optional Valuation Fee. For (+) or (-) difference only.
Example : 20.00 and -20.00
cod_fee Decimal(12,2) Optional Cash-on-Delivery Fee. For (+) or (-) difference only.
Example : 20.00 and -20.00
tfi Decimal(12,2) Optional Temporary Fuel Increase. For (+) or (-) difference only.
Example : 20.00 and -20.00
freight_charge Decimal(12,2) Optional Freight Charge. For (+) or (-) difference only.
Example : 20.00 and -20.00
dst Decimal(12,2) True Documentary Stamp. For (+) or (-) difference only.
Example : 20.00 and -20.00
vat Decimal(12,2) True For (+) or (-) difference only.
Example : 20.00 and -20.00
total_charges_vat_in Decimal(12,2) True For (+) or (-) difference only.
Example : 20.00 and -20.00
total_charges_vat_ex Decimal(12,2) True For (+) or (-) difference only.
Example : 20.00 and -20.00
