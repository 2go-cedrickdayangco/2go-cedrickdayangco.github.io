Compute Rate API - computeRate
Change Logs
Version Description Author
v.20240426

  • Remove w_meals and w_linen as both are required.
 2GO Solutions Delivery
Endpoint URL
GET base_url/computeRate.aspx
Request Parameters
Name Type Required Description
accom String True Accommodation code of the preferred travel. Accommodation value may be derived from getAccom ACCO_CODE
vess_code String True vessel code of the preferred travel. Vessel value may be derived from getSched or getETD VESS_CODE
etd Datetime True Sailing date of the preferred travel. Sailing Date value may be derived from getSched or getETD START_ETD (mm/dd/yyyy hh:mi:ss AM)
orig String True Origin code of the preferred travel. Origin value may be derived from getSched or getETD ORIGIN or getOrigin PORT_CODE
dest String True destination code of the preferred travel. Destination value may be derived from getSched or getETD Destination or getDestination PORT_CODE
w_ins String True Y if with insurance
N if w/out insurance
w_scon String True Y if with TravelLink
N if w/out TravelLink
ooty String True Corresponding OOTY Code of user, value may be retrieved on getLoginDtls user_ooty_code
disc_code String True Discount Code
ND - No Discount
SC - Senior Citizen
ST - Student
INF - INFANT
DABLD - disabled
CHL - child
tID String True Tier ID, value can be retrieved on getBookingClassess tier
bRate Decimal True Base Rate, value can be retrieved on getBookingClassess baserate
trate Decimal True Tier Rate, value can be retrieved on getBookingClassess rate
pax_cat String Optional NEW Passenger Category. Default to "REG" if blank
REG - Regular
UP - Uniformed Personnel
SC - Senior Citizen
PWD - PWD
ST - Student
INF - Infant
pax_id_type String Optional NEW Presented ID relative to passenger category. Required if pax_cat IS NOT REGULAR.
pax_id_no String Optional NEW ID Number of presented ID. Required if pax_cat IS NOT REGULAR.
Sample Request

base_url/computeRate.aspx?accom=SUPER&vess_code=SFX1X&etd=04/14/2024 09:00:00 AM&orig=MNL&dest=CEB&w_ins=Y
&w_scon=N&ooty=ALUZ&disc_code=ND&tID=BRT&bRate=2087.48&trate=1677&pax_cat=sc&pax_id_type=Senior Citizen
&pax_id_no=SC12-23957

 2GO Solutions Delivery
Endpoint URL
POST base_url/saveToEtrans.aspx
Request Parameters
Name Type Required Description
ccode String True Client Code. To be provided by 2GO
ckey String True Client Key. To be provided by 2GO
rese_no String True reservation number, can be retrieved on getReseNo RESERVATION_NO
reit_no String True Reservation item number, count of items related to a single reservation no. A reservation can hold multiple items, these items represents each of the passenger's record.
contact_person String True Name of person w/c the itinerary is billed to.
contact_no String True Contact no. of the contact person w/c the itinerary is billed to.
email String True email address of the contact person w/c the itinerary is billed to.
acco_code String True accommodation code of the preferred travel. Accommodation value may be derived from getAccom ACCO_CODE
pax_type String True Passenger Type options :
B - Bunk
R - Room
M - Minor
F - Free
vess_code String True vessel code of the preferred travel. Vessel value may be derived from getSched or getETD VESS_CODE
voyage_year String True voyage year of the preferred travel. Voyage Year value may be derived from getSched or getETD VOYAGE_YEAR
voyage_no String True voyage number of the preferred travel. Voyage Number value may be derived from getSched or getETD VOYAGE_NO
orig String True origin code of the preferred travel. Origin value may be derived from getOrigin PORT_CODE
dest String True destination code of the preferred travel. Destination value may be derived from getDestination PORT_CODE
etd Datetime True sailing date of the preferred travel. Sailing Date value may be derived from getSched or getETD START_ETD (mm/dd/yyyy hh:mi:ss AM)
eta Datetime True Arrival date of the preferred travel. Arrival Date value may be derived from getSched or getETD END_ETA (mm/dd/yyyy hh:mi:ss AM)
colo_code String True color code, retrievable on getBunks colo_code
cluster_no String True cluster number, retrievable on getBunks cluster_no
alpha String True Alpha Code, retrievable on getBunks alpha
parent_reit String Optional Parent reservation item no. Provide item no. of parent reservation number, applicable for Room accommodation for free passenger and for Minor/Infant Passenger
l_name String True Passenger Last Name - Must match passenger Valid ID
f_name String True Passenger First Name - Must match passenger Valid ID
mid_in String True Passenger Middle Initial
gender String True Passenger gender :
M - Male
F - Female
age String True Passenger Age
address1 String True Passenger Home Address
tID String True Tier ID,value can be retrieved on getBookingClassess
disc_code String True Discount Code
ND - No Discount
SC - Senior Citizen
ST - Student
INF - INFANT
DABLD - disabled
CHL - child
bRate Decimal True Base Rate,value can be retrieved on getBookingClassess
trate Decimal True Tier Rate,value can be retrieved on getBookingClassess
SC_ID String Optional senior citizen ID. Replace w/ “x” if not applicable
co_name String True company name. Replace w/ “x” if not applicable
co_add String True company address. Replace w/ “x” if not applicable
co_tin String True Company TIN. Replace w/ “x” if not applicable
w_ins String True indicate if with insurance or if w/out insurance
pax_cat String Optional NEW Passenger Category. Default to "REG" if blank
REG - Regular
UP - Uniformed Personnel
SC - Senior Citizen
PWD - PWD
ST - Student
INF - Infant
pax_id_type String Optional NEW Presented ID relative to passenger category. Required if pax_cat IS NOT regular.
pax_id_no String Optional NEW ID Number of presented ID. Required if pax_cat IS NOT regular.
