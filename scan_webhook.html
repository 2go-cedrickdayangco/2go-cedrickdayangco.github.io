Request Parameters
Name Type Required Description
tracking_no String True Unique reference string of the transaction. AWB Number
status String True Status Code of the scanning activity. Example : A, H, T
status_date Datetime
(YYYY-MM-DD HH:ii:ss)		 True Date of when the status was done
location String True Location ID of where the status happens (Hub)
origin String Optional Scanning Origin Location
destination String Optional Scanning Destination Location
dlvy_hub String Optional Delivery Hub Store ID
dlvy_city String Optional Required if destination port is Manila.
quantity Number True Number of item inside the box
mps_push String True "Y" or "N". Determine if transaction is mps or not./td>
tracking_dtls Array True Parent Wrapper of the JSON string format for the list of shipment
reference_no String True Shipment Barcode Sticker. Count must be match by the number of quantity
runsheet_number String Optional Runsheet Number
dest_system String true Destination System. Accepted Value: "qair" or "zip"
tracking_url String Optional FarEye Tracking URL
transaction_type String true Transaction Type. Expected values "PX" or "DX"
created_by String Optional Created By
milestone String Optional New Determine status leg. Expected value "FM", "MM" and "LM"
