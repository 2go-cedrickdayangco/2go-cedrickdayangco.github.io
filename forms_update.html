Request Parameters
Name Type Required Description
shipment_form_dtls Shipment form details wrapper.
form_number String(20) True Form Number to be updated
form_status String(6) True Please refer to the Forms Status table for the valid values
Blank / Empty means no changes
date_updated Date
(YYYY-MM-DD HH24:MI:SS) 		True Datetime of when the shipment was updated

Forms Status Table
# Code Description Remarks
1 CLD Cancelled Receipt of complete set of cancelled form
2 DEF Deferred Temporary status of form subject to approval of Controller.
3 REL Released Receipt of forms by company owned stores.
4 USD Used Consumed AWB.
