Endpoint URL
POST TBD
Request Parameters
Name Type Required Description
tracking_no String(50) True AWB Number
customer_name String(255) True Customer Name
delivery_hub String(100) True Delivery Hub
rider_name String(100) True Delivery Rider Name
documents Array True Wrapper for documents
doc_name String(100) True Document Name. Example : Invoice
doc_collected String(1) True Document Status. Example : "Y" if collected. "N" if not yet collected
doc_url String(255) Optional Document URL. Required if document is collected
collected_date Datetime
(YYYY-MM-DD HH24:mm:ss)		 Optional Date of when the document was collected. Required if document is collected
