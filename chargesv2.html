Triggers
AWB Charges API will be triggered 12MN Daily

1. Shipment reaching terminating status : DEL, RTS, RTC, NAC, WFC, Pull-out, LST, PLD + DEL
2. If paymode is "Prepaid", right after encoding.
3. Post-delivery adjustment.
Endpoint URL
POST https://dev01-nrvqudqpv70d-nt.integration.ocp.oraclecloud.com:443/ic/api/integration/v1/flows/rest/FARARCHRGSARRTE/1.0/
