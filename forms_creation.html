Forms Creation API - OICS to Fareye
FarEye Endpoint URL for Forms Creation
Request Parameters
Name Type Required Description
shipment_form_dtls Shipment form details wrapper.
form_type String(5) True Form type: • AWB - Shipment Forms
• CAWB - International forms
• CN - Quiksave
• SRVC - Service mail
• OR - Official Receipts
form_series_start String True Form number start of form series. Example : AA000001A
form_series_end String True Form number end of form series. Example : AA009999A
form_status String True Default to : "REL"
validity_date Date
(YYYY-MM-DD) 		Optional Form validity date. Required if form type is AWB or OR. If blank, forms does not have validity date
released_date Date
(YYYY-MM-DD HH24:MI:SS) 		True Shipment form release date
