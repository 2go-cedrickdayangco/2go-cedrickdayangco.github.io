v.20220524

Adjustment API

FarEye will send Adjustment API whenever a user updates shipment details in FarEeye adjustment module. Adjustment will only be allowed as long the shipment doesn't reached terminating status.
Change Logs
Version Description
v.20220613 Initial Document
Triggers
Upon saving adjustment in FarEye.
Request Parameters
Name Type Required Description
tracking_no String(50) True 2GO Unique reference number. AWB Number of shipment to be updated
adjustment_number String(50) True FarEye generated adjustment number
adjustment_datetime Datetime True Date and time when the adjustment happened
adjustment_type String True NEW Type of Adjustment
adjustment_remarks String True NEW Adjustment remarks
adjustment_by String True UPDATE Previously updated_by User who made the adjustment
declared_value Decimal Optional Updated Declared Value
cod_amount Decimal Optional Updated COD Amount
service_type String(50) Optional Updated Service Type
dest_port String(10) Optional Updated Destination Port
dest_area_code String(10) Optional Updated Destination 2GO Area Code
Name Type Required Description
paymode String(50) Optional Updated Paymode. Allowed for CC and CS only
dest_system String true New Destination System. "qair" or "zip"
