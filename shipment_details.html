Name Type Required Description

pickup_date Date
(YYYY-MM-DD)		 True Pickup Date

billtosite_account String(128) True NEW Fusion Bill To Site Account Number on where the charges should be accounted.
qair_account_number String(128) True QuikAir / ZIP Customer Account Number.
transaction_type String(128) Optional Expected value :
PX - Parcels
DX - Documents
cust_short_name String(128) Optional Phase 2 Customer Short Name must be existing in the Merchant Master.
store_id String(128) Optional Fusion Store Location ID. Inbounding store id
store_name String(128) Optional Fusion Store Location Name
store_type String(128) Optional New Fusion Store Location Type : Agent-owned, 2GO Owned, etc
booking_no String(128) Optional NPR Booking Number
paymode String(128) True Prepaid
Freight Collect
CS - Collect Shipper
CC - Collect Consignee
SC - Service Cargo
service_type String(128) True DD - Door to Door
DP - Door to Pier
PP - Pier to Pier
PD - Pier to Door
AA - Airport to Airport
AD - Airport to Door
DA - Door to Airport
DDS - Door to Door Same Day
ADS - Airport to Door Same Day

shipper_name String(128) True Shipper name in Merchant Master

ultimate_shipper_area_code String(128) Optional 2GO Area Code based on the Ultimate Shipper Address
ultimate_shipper_zip String(128) Optional Ultimate Shipper ZIP
origin_port String(128) True Origin Port
origin_area_code String(128) True Shipper Area Code. It must be existing in Postal code master
origin_landmark String(128) Optional Pickup Location Landmark
cnee_billtosite_account String(128) Optional Fusion Customer Bill to Site Account Number
cnee_name String(128) True Consignee Name
cnee_contact String(128) Optional Consignee Contact Number
cnee_address String(128) True Consignee Complete Address
cnee_address_province String(128) Optional Consignee Province
cnee_address_city String(128) Optional Consignee City
cnee_address_brgy String(128) Optional Consignee Barangay
cnee_address_street String(128) Optional Consignee Address Bldg. / House Number / Street
dest_port String(128) True Destination Port
dest_area_code String(128) True 2GO Area Code based on the Consignee Address
ultimate_cnee String(128) Optional Name of the receiver in case the consignee is not available during delivery
dest_landmark String(128) Optional Consignee Address Landmark
cnee_careoff_store String(128) Optional 2GO Store ID where consignee should pickup the shipment
dest_store_port String(128) Optional Destination Port of the 2GO Store
return_cnee_name String(128) Optional Name of the receiver for return shipment
return_cnee_contact String(128) Optional Contact Number of the receiver for return shipment
return_address String(128) Optional Return complete address
return_address_province String(128) Optional Return Address Province
return_address_city String(128) Optional Return Address City
return_address_brgy String(128) Optional Return Address Barangay
return_address_street String(128) Optional Return Address Bldg. / House Number / Street
return_address_area_code String(128) Optional Return Address Area Code
return_address_port_code String(128) Optional Return Address Port Code
special_instruction String(128) Optional Free text field
attachment String(128) Optional Free text field
or_number String(128) Optional Official Receipt Number. Required. If paymode is "Prepaid"
or_amount Decimal(12,2) Optional Official Receipt Amount. Required. If paymode is "Prepaid"
or_date Date
(YYYY-MM-DD)		 Optional Date of when the Official Receipt was issued. Required. If paymode is "Prepaid"
inbounding_hub String(128) Optional new Inbounding Hub.
dest_hub String(128) Optional new Destination Hub.
cod_amount Decimal(12,2) True Amount to be collected upon delivery. Input "0" for non-cod transaction
declared_value Decimal(12,2) True Declared Value of the shipments
mps_flg Boolean
(Y/N)		 True Flag to indicate if the shipments is Multi-Part Shipment(MPS)
rc_code String(128) True Phase 2 Revenue Center Code
rc_parent_name String(128) True Phase 2 Revenue Center Parent Name
rc_child_name String(128) True Phase 2 Revenue Center Child Name
encoding_date Datetime
(YYYY-MM-DD HH:mm:ss)		 True New Timestamp of when the order was created
item_details Array True Parent wrapper for the item details information
item_no Number True Item Number
item_desc String(128) True UPDATE From item_description to item_desc to align with other API. Item Description
item_qty Number True Item Quantity
package_code String(128) True Fusion Package Code
commodity_code String(128) True Commodity Code must be present in Commodity master list
item_cust_shortname String(128) Optional Phase 2 Per Line Item Customer Short Name
block_measurement Boolean
(Y/N)		 Optional Flag to indicate if the shipments have block measurement
item_weight Decimal(12,2) True Item Actual Weight in KG
item_length Decimal(12,2) True Item Length in cm
item_width Decimal(12,2) True Item Width in cm
item_height Decimal(12,2) True Item Height in cm
weight_vol_kg Decimal(12,2) True NEW Volumetric Weight in kg
weight_vol_cbm Decimal(12,2) True NEW Volumetric Weight in CBM
weight_charge_kg Decimal(12,2) True NEW Chargeable Weight in kg
weight_charge_cbm Decimal(12,2) True NEW Chargeable Weight in CBM
