API Flow
OICS to FarEye
Triggers
Upon inquiry
Endpoint URL
POST To be provided by FarEye
Request Parameters
Name Type Required Description
customer_account_number String Optional Customer Account Number Account Number
type_of_order String(64) True Type of Order : Forward, Reverse
paymode String(64) True Prepaid
Freight Collect
CS - Collect Shipper
CC - Collect Consignee
SC - Service Cargo
service_type String(64) True DD - Door to Door
DP - Door to Pier
PP - Pier to Pier
PD - Pier to Door
AA - Airport to Airport
AD - Airport to Door
DA - Door to Airport
DDS - Door to Door Same Day
ADS - Airport to Door Same Day
origin_zone String(64) True Origin Zone : NCR, Luzon, Visayas, Mindanao
origin_port String(64) True Origin Port
destination_zone String(64) True Destination Zone : NCR, Luzon, Visayas, Mindanao
destination_port String(64) True Destination Port
delivery_category String(64) True Delivery Category : SA, OTD, OSA
cod_amount Decimal Optional Amount to be collected upon delivery
declared_value Decimal True Declared Value of the shipments
item_details Array True Parent wrapper for the item details information
item_no Number True Item Number
item_cust_shortname String(64) Optional Phase 2 Per Line Item Customer Short Name
item_description String(64) True Item Description
item_qty Number True Item Quanity
package_code String(64) True Fusion Package Code
commodity_code String(64) True Commodity Code must be present in Commodity master list
block_measurement (Y/N) Optional Flag to indicate if the shipments have block measurement
item_weight Decimal True Item Actual Weight in KG
item_length Decimal True Item Length
item_width Decimal True Item Width
item_height Decimal True Item Height
chargeable_weight Decimal True Volumetric Weight vs Actual Weight whichever is higher

charge_type String True Charge Type
charge_amount Decimal True Charge Amount
