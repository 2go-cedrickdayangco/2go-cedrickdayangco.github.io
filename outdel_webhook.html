Out for Delivery Updates Webhook - FarEye to OICS
Webhook endpoints for pickup status updates from FarEye to OICS
Endpoint URL
POST TBD
Request Parameters
Name Type Required Description
tracking_no String True Unique reference string of the transaction. AWB Number
transaction_type String true New Transaction Type. Expected values "Px" or "Dx"
status String True Default to "D" Out for Delivery
status_date Datetime
(YYYY-MM-DD HH:ii:ss)		 True Date of when the status was done
runsheet_number String True New Runsheet Number
courier_id String True Courier ID
courier_name String true Courier Name
courier_mobile String true Courier Mobile Number
courier_vehicle_type String true Courier Vehicle Type. Example : Motorcycle, Bike, Multicab
courier_vehicle_plateno String true Plate Number of Courier Vehicle
location String True New Location ID of where the status happens (Hub)
Name Type Required Description
origin String Optional New Scanning Origin Location
destination String Optional New Scanning Destination Location
dlvy_hub String Optional New Delivery Hub Store ID
dlvy_city String Optional New Required if destination port is Manila.
quantity Number True New Number of item inside the box
tracking_dtls Array True New Parent Wrapper of the JSON string format for the list of shipment
reference_no String True New Shipment Barcode Sticker. Count must be match by the number of quantity
dest_system String true New Destination System. Accepted Value: "qair" or "zip"
tracking_url String Optional New FarEye Tracking URL
