    API Flow
    FarEye to OICS
    Triggers
    Cashier recording cash and sf collection from couriers
    Endpoint URL
    POST https://dev01-nrvqudqpv70d-nt.integration.ocp.oraclecloud.com:443/ic/api/integration/v1/flows/rest/FARALSTMLCSHRHOARRTE/1.0/
    Request Parameters
    Name Type Required Description
    reference_no String True FarEye transaction reference number
    store_id String(100) True 2GO Store ID
    store_name String(100) True 2GO Store Name
    store_type String(5) True 2GO Store Type Options : RET, AGT, HUB
    store_city String(100) True 2GO Store City Address
    hub_code String(100) True Hub Code
    dest_port String True Destination Port
    cashier_id String(100) True 2GO Cashier ID
    cashier_name String(100) True 2GO Cashier Name
    rider_id String(100) True 2GO Delivery Rider ID
    tracking_no String True Tracking Number / AWB Number
    shipper_name String True Shipper Name
    cnee_name String True Consignee Name
    paymode String True Transaction Paymode
    cod_payment_mode String Optional COD Payment Mode : cash or cheque
    cod_amount Decimal Optional COD Amount to be collected
    cod_cash_amount Decimal Optional COD Collected Cash
    cod_check_amount Decimal Optional Required if cod payment mode is check COD Check Amount
    cod_check_number String Optional Required if cod payment mode is check COD Check Number
    cod_check_date Date Optional Required if cod payment mode is check COD Check Date
    cod_check_bank_issuer String(100) Optional Required if cod payment mode is check Bank Name of the issuer of check
    sf_payment_mode String Optional Shipping Fee Payment Mode : cash or cheque
    sf_amount Decimal Optional Required if sf payment mode is cash Shipping Fee - Amount to be collected
    sf_cash_amount Decimal Optional Required if sf payment mode is cash Shipping Fee Cash Amount Collected
    sf_check_amount Decimal Optional Required if sf payment mode is check Shipping Fee Check Amount Collected
    sf_check_number String Optional Required if sf payment mode is check Shipping Fee Check Number
    sf_check_date Date Optional Required if sf payment mode is check Shipping Fee Check Date
    sf_check_bank_issuer String(100) Optional Required if sf payment mode is check Bank Name of the issuer of check
    or_number String Optional Required if paymode if Freight Collect O.R. Number
    or_date Date Optional Required if paymode if Freight Collect O.R. Date
    or_amount Decimal Optional Required if paymode if Freight Collect O.R. Amount
    invoice_value Decimal Optional Invoice Value
    date_created Datetime
    (YYYY-MM-DD HH:mm:ss)    		 True Timestamp of when the entry was made
