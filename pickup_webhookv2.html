Pickup Status Updates Webhook - FarEye to OICS



Endpoint URL
POST TBD
Request Parameters
Name Type Required Description
tracking_no String True Unique reference string of the transaction. AWB Number
status String True Status Code of the pickup activity. Example : SERV_OK
status_remarks String optional New
status_date Datetime
(YYYY-MM-DD HH:ii:ss)		 True Date of when the status was done
status_lat Decimal optional New Latitude of where the status happens
status_lng Decimal optional New Latitude of where the status happens
courier_id String True New Courier ID
courier_name String true New Courier Name
courier_mobile String true New Courier Mobile Number
courier_vehicle_type String optional New Courier Vehicle Type. Example : Motorcycle, Bike, Multicab
courier_vehicle_plateno String optional New Plate Number of Courier Vehicle
images Array optional New Required if pickup is success or failed.
Image proof of status, signature or other required documents.
Example : Shipper Signature, Picture of the house.
doc_type string true New Document Type. Example : POD, Invoice, Proof of Status
doc_image string true New URL of the image

Events and Status Mapping

Status Event Description
SERV_OK Pickup Successful The courier successfully picks up the package.
TSND_NEW TO SEND - New Booking Newly created booking for pickup.
TSND_RE TO SEND - Rescheduled Booking Rescheduled booking for pickup.
SENT Courier Dispatch The courier is already dispatched for pickup.
PF_IO Item Oversized Package oversized for pickup.
PF_INR Item Not Ready Item not ready for pickup.
PF_SNA Shipper Not Available Shipper not available at location for pickup.
PF_CBS Cancelled By Shipper Cancelled By Shipper
PF_IPD Incorrect Pickup Details Incorrect pick up details
PF_OC Out of Capacity The courier cannot accomodate to pickup the shipment
PF_FM Force Majeure The courier cannot pickup the shipment due to force majeure
PF_PPK Poor Packaging The courier cannot pickup the shipment due to poor packaging
PF_PG Prohibited Goods The courier did pickup the shipment due to item being prohibited goods
PF_ID Item Damaged The courier did not pickup the shipment due to item damage
