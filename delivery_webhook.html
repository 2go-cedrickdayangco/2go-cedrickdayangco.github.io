Delivery Status Updates Webhook - FarEye to OICS v.20250414
Endpoint URL
POST //dev01-nrvqudqpv70d-nt.integration.ocp.oraclecloud.com:443/ic/api/integration/v1/flows/rest/FARDELSTSUPDTARRTE/1.0/
Request Parameters
Name Type Required Description
tracking_no String True Unique reference string of the transaction. AWB Number
transaction_type String true Transaction Type. Expected values "PX" or "DX"
mps_items Array Optional List of MPS items and their status. Leave blank if shipment is not an MPS
port_sticker String True Port Sticker
port_sticker_status String True Port Sticker Status. Required if shipment is an MPS. Example : PLD, CLD, NAC
port_sticker_status_date Datetime True Port Sticker Status Date. Required if shipment is an MPS
status String True Status Code of the delivery activity. Example : CNA, DEL, PLD, NAC
status_remarks String Optional Required if status is delivery is failed. Rider's remark / additioanl information for failed delivery.
Example : house close, green gate.
status_date Datetime
(YYYY-MM-DD HH:ii:ss)		 True Date of when the status was done
status_lat Decimal optional Latitude of where the status happens
status_lng Decimal optional Latitude of where the status happens
courier_id String True Courier ID
courier_name String true Courier Name
courier_mobile String true Courier Mobile Number
courier_vehicle_type String true Courier Vehicle Type. Example : Motorcycle, Bike, Multicab
courier_vehicle_plateno String true Plate Number of Courier Vehicle
runsheet_number String True Runsheet Number
received_by String optional Required if delivery is successful. Name of person who received the shipment
relationship String optional Required if delivery is successful. Relationship of the receiver to the consignee. Example : Relatives, Neighbor, Guard
id_type String optional Receiver's ID Type. Example : Driver's License, Company ID
id_no String optional Receiver's ID Number
images Array true Image proof of status, signature or other required documents. Example : Consignee Signature, Picture of the house.
doc_type string true Document Type. Example : POD, Invoice, Proof of Status
doc_image string true URL of the image
dest_system String true Destination System. Accepted Value: "qair" or "zip"
rts_account_number String Optional Account Number to be used for RTS transactions. Required if delivery status update is RTs
tracking_url String Optional FarEye Tracking URL
notice_slip String Optional New Customer reference number
failed_reason String Optional New Status Failed Reason
address_type String Optional New Address type, example "Residential"
address_type_values String Optional New Address value, example "House"

Events and Status Mapping
Event Description Triggered By Status
CONSIGNEE NOT AROUND The delivery transport vehicle is out for delivery. Rider - FarEye Mobile CNA
REFUSED TO ACCEPT Consignee refused to accept the package. Rider - FarEye Mobile RTA
REFUSED TO PAY Consignee refused to pay the cash on delivery fee. Rider - FarEye Mobile RTP
FLOODED AREA Area is flooded and the delivery transport vehicle cannot pass through. Rider - FarEye Mobile FLD
BUSINESS CLOSED Consignee office is not functioning anymore. Rider - FarEye Mobile BCL
PAYMENT NOT READY Payment for the cash on delivery (COD) fee is not ready. Rider - FarEye Mobile RDC
TRANSFERRED Consignee transferred to different address. Rider - FarEye Mobile TRF
UNKNOWN CONSIGNEE Consignee is not known in the delivery area. Rider - FarEye Mobile UCN
OFFICE CLOSED Consignee office is already closed. Rider - FarEye Mobile OFC
BAD ADDRESS Unallocated consignee address. Rider - FarEye Mobile BAD
FORCE MAJEURE Force Majeure Rider - FarEye Mobile DFM
DELIVERED Package is delivered successfully. Rider - FarEye Mobile DEL
